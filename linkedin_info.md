# LinkedIn Project: SheGlam

**Project name**
SheGlam: Instant Makeup Services Marketplace

**Description**
Developed a production-grade, end-to-end marketplace platform connecting clients with professional makeup artists. The system features a robust Flask backend with a modular architecture and a secure MongoDB (NoSQL) data layer. By implementing JWT-based stateless authentication with secure `HttpOnly` cookies and CSRF protection, I ensured top-tier security for users and artists.

The platform includes a cinematic, responsive front-end powered by Tailwind CSS and Vanilla JavaScript, offering a seamless booking flow, verified artist portfolios, and real-time status tracking. I also engineered a high-performance administration panel for platform oversight and automated the deployment workflow using Docker and Gunicorn, achieving 100% production readiness.

**Top 5 skills used in this project**
1. Backend Engineering (Flask, Python, & RESTful API Design)
2. Back-End Web Development (MongoDB, Automated Indexing, & Query Optimization)
3. Web Security & Auth (JWT, HttpOnly Cookies, Bcrypt, & CSRF Protection)
4. Front-End Development (Tailwind CSS, Semantic HTML5, & Vanilla JS)
5. System Integration & Workflow Automation (Batch Scripting, Automated Testing, & Production-Ready Standards)
